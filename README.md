# OpenLens Extensions

A collection of community extensions for OpenLens.

---

## Column Resizer Pro

**Finally, resizable columns in OpenLens.**

OpenLens doesn't support column resizing in resource tables out of the box. This extension fixes that.

### Features

- **Resize any column** - Drag the edge of any column header to resize
- **Persistent widths** - Your column sizes are saved and restored across sessions
- **Double-click to auto-fit** - Automatically size columns to fit their content
- **Right-click context menu** - Quick access to auto-fit, reset column, reset all
- **Visual feedback** - Resize indicators, tooltips showing exact width, column highlighting
- **Per-view memory** - Different tables remember their own column widths

### Download

**[Download column-resizer.tar.gz](https://github.com/YakirGiladi/openlens-extensions/raw/main/column-resizer.tar.gz)**

### Installation

1. Download the `.tar.gz` file
2. Open OpenLens
3. Go to **File > Extensions** (Mac: **OpenLens > Extensions**)
4. Drag the `.tar.gz` file into the extensions window, or click "Install" and select the file
5. Restart OpenLens

### Usage

1. Navigate to any resource view (Pods, Deployments, Services, etc.)
2. Hover over the right edge of any column header
3. Drag to resize, or double-click to auto-fit
4. Right-click for more options

### Compatibility

- OpenLens 6.x
